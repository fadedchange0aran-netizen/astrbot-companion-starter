# astrbot_plugin_bookshelf

AstrBot 书架插件。支持上传 TXT/EPUB 书籍、按章节切分存储、阅读指定章节、查看目录、阅读进度、删除书籍、写笔记、看笔记、写读后感、看读后感，以及基于插件页面的共读面板。

> 本目录保留的是适合整合进 starter 的 patch 版。

## 安装

把插件目录放到 AstrBot 的插件路径下，例如：

```bash
cp -r astrbot_plugin_bookshelf /AstrBot/data/plugins/
```

然后在 AstrBot WebUI 插件管理里重载插件，或重启 AstrBot。

## 插件页面

- 插件会在 WebUI 里注册一个 `共读` 页面
- 页面能力包括：
  - 书架列表和切书
  - 页面内上传 `txt/epub`
  - 章节切换和正文阅读
  - 一键把当前章节交给陪读助手，在目标会话里继续聊
- 同一本书发往同一个目标会话时，会记住最近几次主动陪读记录，并同步写入该会话的对话历史，方便后续自然承接

## 指令

```text
/上传书籍 书名 全文
/上传文本 书名
/书架
/目录 书名
/读第 书名 第X章
/阅读进度 书名
/删除书籍 书名
/写笔记 书名 第X章 内容
/看笔记 书名 第X章
/所有笔记 书名
/读后感 书名 第X章 内容
/看读后感 书名 第X章
/共读 书名
```

## 目标会话说明

- 页面里的“呼叫陪读助手讨论”会直接让后端调用模型生成一条可发送消息，再主动发往你填写并保存的目标 `UMO`
- 目标会话 `UMO` 形如 `napcat_qq:FriendMessage:123456789` 或 `napcat_qq:GroupMessage:987654321`
- 当前实现仍然偏向 QQ / NapCat 口径，因此公开版保留 `UMO` 与 `napcat_qq` 示例
